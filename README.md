# 吉時約 ScheduleBooking
吉時約是一個商家行程預約網站，提供商家設定行程、訂單管理功能，讓消費者透過商家官方LINE進行行程預約。

# 主要特色
- 使用LINE OAuth
- 串接LINE LIFF
- 串接金流系統
- 商家行程設定
- 判斷商家行程，已預定反灰鎖定
- 消費者LINE登入及LINE預約行程

# 後端技術
- 環境：Django
- 資料庫：AWS RDS
- 雲端服務：EC2
- NGINX
- Docker
